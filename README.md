# Agrothon Extension

**Agrothon Extension** is **Version 2** of [Agrothon](https://github.com/MiruthyanJayanS/Agrothon) — an AI-powered agricultural modernization project originally built for a **Hackathon at Karpagam College**. This extended version builds on the original with improvements and additional work.

> 📍 This is the **extended/improved version** of the original Agrothon project. See [Agrothon (v1)](https://github.com/MiruthyanJayanS/Agrothon) for the original hackathon submission.

## 📌 About

Like the original, Agrothon Extension focuses on bringing AI-driven tools to agriculture, covering:

1. **Crop disease classification** — identifying diseases from crop images
2. **Crop price prediction** — forecasting prices based on given/historical data
3. **Live soil analysis** — using real-time NPK (Nitrogen, Phosphorus, Potassium) sensor data
4. **Optimal crop suggestion** — recommending the best crop to grow based on current soil conditions

This version extends the original project with refinements and additional functionality built after the initial hackathon submission.

## ✨ Features

- 🌿 **Crop Disease Classification** — identifies diseases from crop images
- 💰 **Crop Price Prediction** — predicts crop prices based on provided data
- 🧪 **Live Soil Analysis** — integrates real-time NPK sensor data to assess soil health
- 🌱 **Optimal Crop Suggestion** — recommends the best crop based on current soil conditions
- 🔧 Improvements and additions beyond the original v1 implementation

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- Required ML libraries (see `requirements.txt`)
- NPK sensor hardware setup (for live soil data — optional if using sample/test data)

### Setup

Clone the repository and install the required modules:
```bash
git clone https://github.com/MiruthyanJayanS/Agrothon_extension.git
cd Agrothon_extension
pip install -r requirements.txt
```

### Training the Model
```bash
python train_bot.py
```

### Running the App
```bash
streamlit run bot.py
```
